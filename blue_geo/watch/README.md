# `@geo watch`

`@geo watch` watches a target on Earth.,

```bash
 > @geo watch help
@geo watch \
	[dryrun,upload] \
	[target=chilcotin-river-landslide,datetime=<2024-07-30/2024-08-15>,lat=<51.83>,lon=<-122.78>] \
	[dryrun,to=aws_batch|generic|local] \
	[-|<object-name>]
 . watch a target.
```
